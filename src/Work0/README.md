# CG-Lab 万有引力粒子群仿真项目

基于 **Taichi GPU 并行计算** 的万有引力粒子群仿真系统，实现粒子在二维空间中的引力相互作用，并提供实时可视化与交互。

## 一、项目架构

本项目采用 **Python 标准 src 布局**，目录结构如下：

    CG-Lab/
    │
    ├── src/                    # 源代码目录
    │   └── Work0/              # 第0次实验模块
    │       ├── __init__.py     # 包初始化文件
    │       ├── config.py       # 配置文件（粒子数量、引力参数等）
    │       ├── physics.py      # 物理计算（万有引力公式 + GPU并行）
    │       └── main.py         # 主程序（渲染、交互、入口）
    │
    ├── .gitignore              # Git 忽略规则
    └── README.md               # 项目说明文档


## 二、运行方式

在项目根目录运行：

    uv run -m src.Work0.main

即可启动粒子群仿真程序。


## 三、实现功能

- 基于 GPU 并行计算的万有引力粒子群仿真  
- 实时可视化粒子运动效果  
- 支持鼠标交互  
- 模块化设计，代码结构清晰，便于扩展
