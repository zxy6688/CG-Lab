# Work2 - Bezier Curve

计算机图形学实验二：**贝塞尔曲线（Bezier Curve）**

本实验基于 Taichi 实现一个交互式贝塞尔曲线绘制程序。用户可以通过鼠标点击添加控制点，程序根据控制点实时生成并显示贝塞尔曲线，同时绘制控制点及控制折线，便于观察曲线的生成过程。

## 实验内容

本实验的主要任务是实现贝塞尔曲线的交互式绘制，具体包括：

- 鼠标点击添加控制点
- 显示控制点与控制折线

## 实现思路

本程序采用 **de Casteljau 算法** 计算贝塞尔曲线上的点。

对于给定的一组控制点，程序在参数区间 $t \in [0,1]$ 上进行均匀采样。对每一个采样值 $t$，通过 de Casteljau 算法逐层线性插值，最终得到曲线上的一个点。将所有采样点连接起来，就可以近似得到整条贝塞尔曲线。

为了提高显示效率，程序将曲线采样点统一存入缓冲区，再交由 GPU 进行绘制。这样能够减少重复的数据传输和逐点绘制带来的开销，使交互过程更加流畅。

与参考代码相比，本实现对程序结构和交互方式进行了调整，例如加入了撤销控制点、调节采样精度等功能，并采用了不同的代码组织方式，使程序更符合自主实现的风格。

## 交互说明

程序启动后，会弹出图形窗口。可以进行如下操作：

- **鼠标左键**：添加控制点
- **鼠标右键**：撤销最后一个控制点

## 核心算法说明

贝塞尔曲线的计算基于控制点的递推插值。

设控制点为 $P_0, P_1, \dots, P_n$，则 de Casteljau 算法通过不断对相邻点进行线性插值，逐步得到更高层的中间点，最终得到曲线上在参数 $t$ 处对应的点。

线性插值公式为：

$$
P_i^{(k)}(t) = (1-t)P_i^{(k-1)}(t) + tP_{i+1}^{(k-1)}(t)
$$

其中：

- $P_i^{(0)} = P_i$ 表示原始控制点
- 经过多轮递推后，最终得到曲线点 $P_0^{(n)}(t)$

该方法具有良好的几何直观性，并且数值稳定，适合教学实验中实现贝塞尔曲线。
